## Overview
**PowerRoster** is a **desktop client management application** for **freelance personal fitness trainers**.

It helps trainers **manage their client roster** by keeping important information such as contact details, notes, dietary restrictions, injuries, and training locations in a **single centralized application**, optimised for text commands.

## Main Features
The main features of the app are:
- List all clients in the roster
- Add a client and the relevant information
- Delete a client
- Attach notes to a client
- Search a client by name
- Filter the client roster by gym location

Note: this project is still under development and the features and documentation will be improved along the way.
